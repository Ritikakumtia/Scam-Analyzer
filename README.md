# Scam Analyzer

A web-based application that analyzes suspicious URLs, messages, and online content to identify potential scams and fraudulent activities. The project helps users assess risks by leveraging machine learning, text analysis, and web technologies.

## 🚀 Features

- URL Scam Detection
- Suspicious Content Analysis
- Risk Score Generation
- User-Friendly Dashboard
- Real-Time Results
- Responsive Frontend
- API Integration
- Detailed Scam Reports

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Ritikakumtia/Scam-Analyzer.git
cd Scam-Analyzer
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Machine Learning Setup

```bash
pip install -r requirements.txt
python train_model.py
```

## 🔍 Working Flow

1. User submits URL or text content.
2. Frontend sends request to backend API.
3. Backend processes the input.
4. Machine Learning model analyzes risk factors.
5. Risk score and scam prediction are generated.
6. Results are displayed on the dashboard.
